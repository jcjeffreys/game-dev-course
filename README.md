# game-dev-course
Resources for teaching game development.

## Contents

### PICO-8 Starter Course

A complete beginner-friendly course for learning game development using PICO-8, a fantasy console perfect for beginners. The course includes:

- **5 Progressive Lessons**: From "Hello World" to a complete game
- **Detailed Comments**: Every line explained for learning
- **Hands-on Examples**: Interactive programs you can modify and experiment with
- **Challenges**: Extend your learning with suggested modifications

#### What You'll Learn

1. **Lesson 1: Hello World** - Program structure and the game loop
2. **Lesson 2: Drawing Shapes** - Graphics, coordinates, and colors
3. **Lesson 3: Player Movement** - Input handling and interactivity
4. **Lesson 4: Bouncing Ball** - Physics and collision detection
5. **Lesson 5: Collecting Items** - Complete game with scoring and win conditions

Each lesson builds on the previous one, teaching fundamental game development concepts that apply to any game engine or platform.

## Getting Started

1. Download PICO-8 from [lexaloffle.com/pico-8.php](https://www.lexaloffle.com/pico-8.php)
2. Navigate to the [pico8](pico8/) directory
3. Follow the course in order, starting with the README
